How to manage a sales pipeline in Gmail

Gmail is where your sales conversations happen. But it has no concept of a sales pipeline — no deal stages, no values, no way to see which opportunities are moving and which are stuck. Here's how to add that layer.

Method 1: Gmail labels as pipeline stages

The free, built-in approach:

  1. Create Gmail labels for each stage: Lead, Contacted, Proposal, Negotiation, Won, Lost
  2. Apply the appropriate label to each deal thread
  3. Use color-coding to make stages visually distinct
  4. Click a label to see all deals in that stage

Limitation: Labels show you threads, not pipeline metrics. You can't see total deal value, days in stage, or which deals are going stale. And you have to remember to update labels manually — which stops happening after the first busy week.

Method 2: Gmail + a sidebar CRM

Tools like HubSpot and Copper add a sidebar panel to Gmail that shows CRM data alongside your email. You can update deal stages from the sidebar without leaving Gmail.

Limitation: The CRM still lives somewhere else — the sidebar is a window into it. You still need to maintain the CRM, and you're paying for a full platform when you might only need the pipeline view.

Method 3: A Gmail-native pipeline tool

Extensions like Streak and NeverMiss add pipeline features directly to Gmail. Streak offers a full CRM with customizable pipeline views. NeverMiss offers a compact pipeline summary plus deal context on every thread.

The advantage: deal data is attached to email threads, so the pipeline updates as you work instead of requiring a separate data entry step.

What you actually need from a Gmail pipeline

  • Stage per thread — know at a glance whether this is a lead, a proposal, or a negotiation
  • Deal value — understand what's at stake in each conversation
  • Next action date — when is the next follow-up due?
  • Stale detection — which deals have gone quiet without you noticing?
  • Summary view — how many active deals, total pipeline value, what needs attention today?
